Aquest programa analitza un String i separa les lletres i els numeros.
-- Les lletres
Agafa les lletres individualment, les compara entre elles i recull en un String les lletres úniques.
A partir d'aquest String el compara amb l'antic i realitza un reocompte de les lletres repetides.

--Els números
Agafa els caràcter i els passa per un métode que els intenta passar a Enter, si aquest resulta envia un paràmetre Boleá al programa que el permet contar, si aquest surt erroni el programa el descarta con a número i el passa per el contador de les lletres.
